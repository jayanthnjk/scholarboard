# Multi-Tenant SaaS ERP UI - Design System

Welcome to the component documentation for the Multi-Tenant SaaS ERP UI platform.

## Structure

Components are organized into the following categories:

- **UI** – Base design system components (Button, Input, Modal, etc.)
- **Layout** – Page structure components (PageHeader, Sidebar, Grid, etc.)
- **Composite** – Higher-order composed components (DataTable, FormField, StatCard)
- **Module** – Module-specific components (StudentList, FeeStructure, etc.)

## Design Tokens

The platform uses CSS custom properties for theming, supporting:
- Light and dark modes
- Tenant-specific branding colors
- RTL/LTR layout switching
- High-contrast accessibility mode

## Accessibility

All components target WCAG 2.1 AA compliance with:
- Proper ARIA attributes and roles
- Keyboard navigation support
- Focus management
- Screen reader compatibility
